# 简介
这是一个计算玩家**本月可用黄票换多少抽**的程序，使用**Visual Studio 2022**进行编写。*因为是萌新初学C敲出来的屑作，只能在控制台上看，不带UI。*:(
## 使用方法
按照程序的提示输入您**目前的黄票数**。程序默认循环运行，输入非数字字符以结束程序。
## 目前程序可显示的内容
* 目前黄票总共可兑换的抽数
* 可换取的单抽数和十连数
* 兑换后剩余的黄票数
* 一个循环结束后会根据`printf()`的返回值打印分割线
